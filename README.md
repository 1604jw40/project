# backend-study

백엔드 학습을 위한 JavaScript, Node.js, Express, TypeScript, Database, Auth 학습 프로젝트다.

최종 목표는 Express + TypeScript + PostgreSQL + Prisma + JWT 기반의 백엔드 프로젝트를 완성하는 것이다.

## 현재 진행 상태

### 완료

- 01-javascript/01-syntax-basic
- 01-javascript/02-variable-safety

### 현재 학습 위치

- JavaScript 기본기 보강
- `let`, `const`, 블록 스코프, 객체 응답 형태 학습 완료

## 학습 로드맵

1. JavaScript 기본기
2. Node.js 런타임
3. 백엔드 기본 개념
   - HTTP
   - REST API
   - JSON
   - Request / Response
   - Status Code
   - Error Handling
4. Express
5. TypeScript
6. Express + TypeScript
7. Database
   - SQL
   - PostgreSQL
   - Prisma
8. Auth
   - Password Hashing
   - JWT
   - Auth Middleware
9. Testing
10. Harness Engineering
11. Final Project

## 진행 원칙

- 필요한 디렉토리만 단계적으로 생성한다.
- 각 챕터는 작은 실습 단위로 진행한다.
- 챕터가 끝나면 해당 README를 갱신한다.
- 백엔드와 연결되는 JavaScript 사용법을 우선적으로 학습한다.
- 최종 프로젝트는 MVP → 확장 기능 → 테스트 순서로 진행한다.
