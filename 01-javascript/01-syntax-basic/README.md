# 01 Syntax Basic

JavaScript의 기본 문법을 학습한 챕터다.

## 학습한 내용

- `console.log()`로 출력하기
- 변수 만들기
- 기본 자료형 사용하기
- 산술 연산자와 비교 연산자 사용하기
- 조건문 사용하기
- 반복문 사용하기
- 함수 만들기
- 배열 사용하기
- 객체 사용하기

## 예제 파일

```txt
examples/hello.js
examples/variables.js
examples/operators.js
examples/conditions.js
examples/loops.js
examples/functions.js
examples/arrays.js
examples/objects.js
```

## 실습 파일

## 완료 기준

- JavaScript 파일을 Node.js로 실행할 수 있다.
- 변수, 조건문, 반복문, 함수, 배열, 객체를 간단히 사용할 수 있다.
- 배열 안의 객체를 반복하면서 조건에 맞는 값을 출력할 수 있다.
